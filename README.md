Batagor Kang Bob – Website Warung Makanan Kecil

Website sederhana untuk mempromosikan **Batagor Kang Bob**, sebuah warung batagor legendaris yang sudah berdiri sejak 2010. Dibuat menggunakan **HTML, CSS, dan JavaScript** dengan tampilan modern sederhana namun tetap ringan.

🖼️ Fitur Utama

- Halaman beranda dengan deskripsi usaha
- Daftar menu batagor dengan harga dan gambar
- Google Maps lokasi warung
- Responsive design (mobile friendly)

🧑‍💻 Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (vanilla)
- [Google Fonts](https://fonts.google.com/) – untuk gaya teks
- [Bootstrap Icons](https://icons.getbootstrap.com/) – untuk ikon
